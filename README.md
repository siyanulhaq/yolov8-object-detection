# 🎯 Real-Time Object Detection using YOLOv8

A real-time object detection project using YOLOv8 and OpenCV
that detects 80 common objects through a webcam.

## 🚀 Demo
Show any object to your webcam and it will detect it instantly!

## 🛠️ Tech Stack
- YOLOv8 (Ultralytics)
- OpenCV
- Python
- Roboflow (dataset)
- Google Colab (training)

## 📦 Installation
pip install ultralytics opencv-python

## ▶️ Run
python detect.py

## 🎯 Objects it can detect (80 classes)
person, car, phone, laptop, bottle, cup, scissors,
apple, banana, dog, cat, and 70+ more!

## 📊 Dataset
Trained on COCO Dataset via Roboflow
